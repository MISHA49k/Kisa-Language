# 🎮 Команды для создания игр в Kisa Language

## Обзор игровых команд

Kisa Language теперь поддерживает создание простых текстовых игр с использованием специальных команд.

---

## 📋 Список всех игровых команд

### 1. **ИГРА** - Инициализация игры
Создает новую игру с заданными параметрами.

```kisa
игра название:"Название игры" ширина:800 высота:600.
```

**Параметры:**
- `название` - название игры (строка)
- `ширина` - ширина экрана (число, по умолчанию 800)
- `высота` - высота экрана (число, по умолчанию 600)

---

### 2. **СЦЕНА** - Создание сцены
Добавляет сцену в игру с заданным сценарием.

```kisa
сцена название_сцены сценарий:{
  показать заголовок Добро пожаловать в игру.
  показать сообщение Нажми кнопку для начала.
}.
```

**Параметры:**
- `название_сцены` - название сцены (идентификатор)
- `сценарий` - команды, которые выполняются при загрузке сцены

---

### 3. **ОБЪЕКТ** - Создание игрового объекта
Создает объект в игре (персонаж, враг, предмет и т.д.).

```kisa
объект герой позиция:(100,100) размер:(50,50) цвет:синий.
объект враг позиция:(500,300) размер:(40,40) цвет:красный изображение:"враг.png".
```

**Параметры:**
- `имя_объекта` - имя объекта (идентификатор)
- `позиция` - координаты (x,y) в формате `(число,число)`
- `размер` - размер объекта (ширина,высота) в формате `(число,число)`
- `цвет` - цвет объекта (название цвета или RGB)
- `изображение` - путь к изображению (опционально)

---

### 4. **СОБЫТИЕ** - Регистрация событий
Добавляет обработчик события игры.

```kisa
событие движение при нажатии стрелка_вверх.
событие столкновение объект:враг действие:урон.
```

**Параметры:**
- `тип_события` - тип события (`движение`, `столкновение`, и т.д.)
- `при нажатии` - триггер события (клавиша)
- `объект` - объект, участвующий в событии
- `действие` - действие при наступлении события

---

### 5. **НАЧИСЛИ** - Изменение статистики
Увеличивает или уменьшает характеристику игрока.

```kisa
начисли очки плюс:10.
начисли здоровье минус:5.
начисли энергия плюс:20.
```

**Параметры:**
- `тип_статистики` - название статистики (`очки`, `здоровье`, `энергия` и т.д.)
- `плюс` или `минус` - операция увеличения/уменьшения
- `число` - значение из��енения

---

### 6. **ОБНОВИ** - Обновление свойств объекта
Изменяет свойства существующего объекта.

```kisa
обнови герой позиция:(150,100) скорость:5.
```

**Параметры:**
- `имя_объекта` - имя объекта для обновления
- `позиция` - новые координаты (опционально)
- `скорость` - новая скорость (опционально)
- и другие свойства

## 🎯 Примеры игр

### Пример 1: Простая игра с героем

```kisa
игра название:"Приключения" ширина:800 высота:600.

показать заголовок Начало игры.

объект герой позиция:(100,100) размер:(50,50) цвет:синий.
объект враг позиция:(700,100) размер:(40,40) цвет:красный.

показать сообщение Вам нужно победить врага.

начисли очки плюс:0.
начисли здоровье плюс:100.
```

### Пример 2: Игра с событиями

```kisa
игра название:"Квест" ширина:1024 высота:768.

сцена главная сценарий:{
  показать заголовок Главная сцена.
  объект лут позиция:(500,400) размер:(30,30) цвет:желтый.
  показать сообщение Найди предмет.
}.

событие движение при нажатии стрелка_вверх.
событие столкновение объект:лут действие:подобрать.

начисли очки плюс:100.
```

### Пример 3: Система очков

```kisa
игра название:"Таймер" ширина:640 высота:480.

объект игрок позиция:(320,240) размер:(50,50) цвет:зеленый.

начисли очки плюс:50.
показать сообщение Первое испытание завершено.

начисли очки плюс:30.
показать сообщение Второе испытание завершено.

начисли здоровье минус:10.
показать сообщение Вы ранены.
```

### Пример 4: Динамическое обновление объекта

```kisa
игра название:"Движение" ширина:800 высота:600.

объект персонаж позиция:(50,50) размер:(50,50) цвет:синий.

показать сообщение Персонаж движется.

обнови персонаж позиция:(100,100) скорость:10.

показать сообщение Персонаж достиг новой позиции.
```

## 💡 Советы по использованию

### 1. Структура игры
```kisa
# 1. Инициализация игры
игра название:"Моя игра" ширина:1280 высота:720.

# 2. Создание сцен
сцена меню сценарий:{...}.
сцена игра сценарий:{...}.

# 3. Создание объектов
объект герой позиция:(640,360) размер:(50,50) цвет:синий.

# 4. Регистрация событий
событие движение при нажатии стрелка_вверх.

# 5. Обновление игрового состояния
начисли очки плюс:10.
```

### 2. Хорошие практики
- Используйте понятные имена для объектов
- Группируйте связанные команды
- Добавляйте сообщения для пользователя (`показать сообщение`)
- Управляйте статистикой игрока с помощью `начисли`

### 3. Комбинирование с базовыми командами
```kisa
переменная очки_игрока тип:число значение:0.

если очки_игрока равно 100.
  показать сообщение Вы победили!
конец.
```

## 📚 Дополнительные ресурсы

Основные команды Kisa Language:
- `выведи` - вывод текста
- `показать` - вывод форматированного текста
- `переменная` - объявление переменной
- `если`...`иначе` - условное выполнение
- `функция` - определение функции
- `начисли` - изменение статистики (новое)
- `объект` - создание объекта (новое)
