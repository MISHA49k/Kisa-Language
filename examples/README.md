## 📚 Описание примеров

### 0️⃣ **Hello World** (`0_hello_world.py`)

**Уровень:** Начинающий  
**Описание:** Основная программа для знакомства с Kisa Language  
**Демонстрирует:**
- Вывод текста (`выведи`, `показать`)
- Создание переменных
- Простые команды

**Запуск:**
```bash
python examples/0_hello_world.py
```

### 1️⃣ **Приключение Героя** (`1_simple_adventure.py`)

**Уровень:** Средний  
**Описание:** Простая приключенческая игра с боевой системой  
**Демонстрирует:**
- Создание игры (`игра`)
- Создание игровых объектов (`объект`)
- Движение объектов (`обнови`)
- Система здоровья и энергии (`начисли`)
- Повествование (сюжет)

**Сюжет:**
1. Герой видит врага и сундук
2. Герой движется вперед
3. Находит монету
4. Враг атакует
5. Герой побеждает врага
6. Герой берет сокровище

**Запуск:**
```bash
python examples/1_simple_adventure.py
```

### 2️⃣ **Система Очков** (`2_scoring_system.py`)

**Уровень:** Средний  
**Описание:** Игра с системой прогрессии и сбором очков  
**Демонстрирует:**
- Система очков (`начисли очки`)
- Несколько испытаний
- Прогрессирующая сложность
- Финальный подсчет

**Испытания:**

2. Беговая дорожка (+40 очков)
3. Боевой дух (+60 очков)
4. Собиратель (+80 очков)

**Итого:** 205 очков

**Запуск:**
```bash
python examples/2_scoring_system.py
```

### 3️⃣ **Квест за Артефактом** (`3_multi_scene_quest.py`)

**Уровень:** Продвинутый  
**Описание:** Многосцённая приключенческая игра  
**Демонстрирует:**
- Несколько сцен (Лес, Гора, Подземелье)
- Система здоровья (урон и восстановление)
- Система артефактов (сбор предметов)
- Классическая структура квеста
- Сложные враги (лесной дух, дракон, монстр)

**Сцены:**
1. **Лес** - встреча с лесным духом, получение артефакта
2. **Гора** - битва с драконом, получение артефакта
3. **Подземелье** - встреча с монстром, получение артефакта
4. **Возвращение** - путь домой и финальные титры

**Здоровье:**
- Начальное: 100
- После леса: 90 (-10)
- После горы: 60 (-30)
- После подземелья: 40 (-20)

**Запуск:**
```bash
python examples/3_multi_scene_quest.py
```

## ✏️ Как создать свой пример

1. Создайте новый файл: `examples/X_your_game.py`
2. Скопируйте структуру из одного из примеров
3. Напишите свой код на Kisa Language
4. Запустите: `python examples/X_your_game.py`

**Пример структуры:**

```python
#!/usr/bin/env python3
"""
Описание вашей игры
"""

from kisa_interpreter import run_kisa_code

game_code = """
игра название:"Название" ширина:800 высота:600.

ваш код здесь...
"""

if __name__ == "__main__":
    print("заголовок")
    result = run_kisa_code(game_code)
    print(result)
```

## 🎮 Создание собственной игры

### Шаблон базовой игры

```kisa
# 1. Инициализация
игра название:"Название игры" ширина:800 высота:600.

# 2. Информация для игрока
показать заголовок Название
показать сообщение Описание

# 3. Создание объектов
объект герой позиция:(100,100) размер:(50,50) цвет:синий.
объект враг позиция:(700,100) размер:(40,40) цвет:красный.

# 4. Инициализация статистики
начисли здоровье плюс:100.
начисли очки плюс:0.

# 5. Основной игровой процесс
показать сообщение Начало игры!
обнови герой позиция:(200,100).

# 6. Финал
показать заголовок Конец игры!
```

## 🔧 Полезные команды

```kisa
# Вывод
выведи тип:текст Текст
показать сообщение Текст
показать заголовок Текст

# Создание объектов
объект имя позиция:(x,y) размер:(w,h) цвет:цвет.

# Обновление
обнови имя позиция:(x,y) скорость:5.

# Статистика
начисли очки плюс:10.
начисли здоровье минус:5.

# Проверки
проверить x равно 100.
если x равно 100. ... конец.
```

## 🚀 Советы для создания интересных игр

1. **Используйте истории** - добавьте повествование через `показать сообщение`
2. **Создавайте препятствия** - используйте врагов и урон
3. **Награждайте игрока** - давайте очки и артефакты
4. **Добавляйте прогрессию** - усложняйте игру по ходу
5. **Делайте ясный финал** - показывайте результат и благодарность
