# Язык программирования Kisa 🐱

Простой и элегантный язык программирования на русском языке для работы с текстом и веб-элементами.

## Возможности

- ✅ Полностью на русском языке
- ✅ Интерпретатор на Python
- ✅ Простой и понятный синтаксис
- ✅ Поддержка текста и переменных
- ✅ Создание HTML элементов (ссылки)

## Синтаксис

### Основные команды

#### 1. `выведи` - Вывод текста
```kisa
выведи тип:текст Привет.
выведи тип:текст Hello World.
```

#### 2. `распознать` - Определение типа переменной
```kisa
распознать тип:имя как текст.
```

#### 3. `сделай` - Создание элементов
```kisa
сделай текст ссылку название:YouTube, переход при нажатии:сайт ссылка:youtube.com.
```

### Использование переменных

Переменные указываются в скобках `(переменная)`:

```kisa
выведи тип:текст Привет, (имя).
```

## Установка и запуск

### Требования
- Python 3.7+

### Запуск интерпретатора

```bash
# Создать файл программы
echo 'выведи тип:текст Привет.' > program.kisa

# Запустить интерпретатор
python3 main.py program.kisa
```

### Использование как модуля Python

```python
from kisa_interpreter import run_kisa_code

code = """
выведи тип:текст Привет.
распознать тип:имя как текст.
выведи тип:текст Привет, (имя).
выведи тип:текст Предлагаю.
сделай текст ссылку название:YouTube, переход при нажатии:сайт ссылка:youtube.com.
"""

result = run_kisa_code(code)
print(result)
```

## Примеры программ

### Пример 1: Приветствие

## Архитектура

1. **Lexer (kisa_lexer.py)** - Преобразует исходный код в токены
2. **Parser (kisa_parser.py)** - Преобразует токены в AST (Abstract Syntax Tree)
3. **Interpreter (kisa_interpreter.py)** - Исполняет AST и выдаёт результат

## Команды

### Команда `выведи`
Выводит текст на экран.

**Синтаксис:**
```kisa
выведи тип:текст <значение>.
```

**Примеры:**
```kisa
выведи тип:текст Привет.
выведи тип:текст Привет, (имя).
```

### Команда `распознать`
Определяет переменную и её тип.

**Синтаксис:**
```kisa
распознать тип:<имя> как <тип>.
```

**Примеры:**
```kisa
распознать тип:имя как текст.
распознать тип:возраст как число.
```

### Команда `сделай`
Создаёт HTML элементы (ссылки, тексты и т.д.).

**Синтаксис:**
```kisa
сделай <тип> <элемент> <параметр>:<значение>, <параметр>:<значение>.
```

**Примеры:**
```kisa
сделай текст ссылку название:YouTube, ссылка:youtube.com.
сделай текст ссылку название:GitHub, ссылка:github.com.
```

## Расширение функционала

Для добавления новых команд:

1. Добавьте тип токена в `TokenType` в `kisa_lexer.py`
2. Добавьте ключевое слово в словарь `keywords` в классе `Lexer`
3. Создайте новый класс команды в `kisa_parser.py`
4. Добавьте метод парсера `parse_*_command`
5. Добавьте метод выполнения в `KisaInterpreter`
