> **Kisa** - это инновационный язык программирования на русском языке, разработанный для обучения и создания игр. Простой синтаксис, мощные возможности, поддержка русскоязычных команд.

---

## ✨ Особенности

### 🎯 Русскоязычный синтаксис
```kisa
выведи тип:текст Привет, мир!
переменная имя тип:текст значение:Иван.
```

### 🎮 Встроенная поддержка игр
```kisa
игра название:"Приключения" ширина:800 высота:600.
объект герой позиция:(100,100) размер:(50,50) цвет:синий.
начисли очки плюс:10.
```

### 🔧 Полнофункциональность
- ✅ Переменные и типизация
- ✅ Функции и процедуры
- ✅ Условные операторы

- ✅ Работа с файлами
- ✅ Обработка ошибок
- ✅ Создание игр
- ✅ HTML-элементы

## 🚀 Быстрый старт

### Установка

```bash
git clone https://github.com/MISHA49k/Kisa-Language.git
cd Kisa-Language
python kisa_interpreter.py
```

### Первая программа

```kisa
показать заголовок Добро пожаловать в Kisa.
выведи тип:текст Это мой первый язык программирования!
```

### Создание простой игры

```kisa
игра название:"Моя первая игра" ширина:800 высота:600.

объект герой позиция:(100,100) размер:(50,50) цвет:синий.
объект враг позиция:(700,100) размер:(40,40) цвет:красный.

показать сообщение Игра запущена! Победи врага!

начисли очки плюс:100.
начисли здоровье плюс:50.
```

## 📚 Примеры кода

### Переменные и операции

```kisa
переменная x тип:число значение:10.
переменная y тип:число значение:20.
переменная сумма тип:число значение:30.

выведи тип:текст Результат: 30.
```

### Функц��и

```kisa
функция приветствие параметры:(имя).
  выведи тип:текст Привет, (имя)!
конец.

вызов приветствие.
```

### Условные операторы

```kisa
переменная возраст тип:число значение:18.

если возраст равно 18.
  показать сообщение Ты совершеннолетний!
иначе.
  показать сообщение Ты еще молодой.
конец.
```

### Работа с файлами

```kisa
переменная данные тип:текст значение:Важная информация.
сохранить данные в файл "сохранения.json".
```

## 🎯 Архитектура

```
Kisa Language
├── kisa_lexer.py       📝 Лексический анализ (токенизация)
├── kisa_parser.py      🌳 Синтаксический анализ (AST)
├── kisa_interpreter.py ⚙️  Интерпретация и выполнение
└── GAME_COMMANDS.md    🎮 Документация игровых команд
```

### Процесс выполнения

```
Исходный код
     ↓
  Лексер (kisa_lexer.py)
     ↓
   Токены
     ↓
  Парсер (kisa_parser.py)
     ↓
    AST
     ↓
 Интерпретатор (kisa_interpreter.py)
     ↓
  Результат
```

## 📖 Документация

### Основные команды

| Команда | Описание | Пример |
|---------|---------|--------|
| `выведи` | Вывод текста | `выведи тип:текст Привет!` |
| `показать` | Форматированный вывод | `показать сообщение Готово!` |
| `переменная` | Объявление переменной | `переменная x тип:число значение:10.` |
| `если` | Условный оператор | `если x равно 10. ... конец.` |
| `функция` | Определение функции | `функция имя параметры:(p). ... конец.` |

### Игровые команды

| Команда | Описание | Пример |
|---------|---------|--------|
| `игра` | Инициализация игры | `игра название:"Игра" ширина:800 высота:600.` |
| `сцена` | Создание сцены | `сцена главная сценарий:{...}.` |
| `объект` | Создание объекта | `объект герой позиция:(100,100) размер:(50,50) цвет:синий.` |
| `начисли` | Изменение статистики | `начисли очки плюс:10.` |
| `обнови` | Обновление объекта | `обнови герой позиция:(150,100).` |

**📚 Полная документация:** [GAME_COMMANDS.md](./GAME_COMMANDS.md)

```kisa

показать заголовок Начало эпической игры.

показать сообщение Победи врага и спаси королевство!

начисли здоровье плюс:100.
```

```kisa
игра название:"Таймер" ширина:640 высота:480.

начисли очки плюс:50.
показать сообщение Первое испытание завершено!

начисли очки плюс:30.
показать сообщение Второе испытание завершено!

начисли здоровье минус:10.
```
